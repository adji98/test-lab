1. Install Go ( https://golang.org/dl/ )

2. Install Buffalo ( https://gobuffalo.io/en/docs/getting-started/installation )

3. Install PostgreSQL (https://www.postgresql.org/download/)

4. Extract the source codes inside the special folder. To reach this folder you need to first open Go's folder (usually it's %HOME%/go), then open src folder, then github.com folder, then open the intan.sari53 folder. Extract the source codes inside this intan.sari53 folder

5. After extract, browse to the streaming_web folder, run "buffalo dev", the Web now open at port 3000

6. Still inside intan.sari53 folder, open the streaming_web_api folder

7. Execute "buffalo db migrate up"

8. Execute "buffalo dev", the API is now running at port 3001
